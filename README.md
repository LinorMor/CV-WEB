# Linor Mor — CV Website

A luxurious, animated one-page CV built with **Next.js (App Router) + React +
Framer Motion**. Dark navy + champagne-gold aesthetic, scroll-driven career
timeline, animated stat counters, English/Hebrew toggle with full RTL, and
downloadable résumé (EN + HE).

## Run locally

```bash
npm install
npm run dev      # http://localhost:3000
```

## Build for production

```bash
npm run build    # next build
npm run start    # serve the production build locally
```

## Deploy to Vercel

This is a standard Next.js app — Vercel auto-detects it.

1. Push this repo to GitHub (or GitLab/Bitbucket).
2. On [vercel.com](https://vercel.com) → **New Project** → import the repo.
3. Framework preset: **Next.js** (auto-detected). No config needed — just **Deploy**.

The home page is statically prerendered (`○ Static`), so it serves as fast,
cached static content on Vercel's edge. You can also wire a custom domain in
the Vercel dashboard.

## Project structure (App Router)

```
src/
  app/
    layout.jsx       Root layout — fonts (next/font), <html>, metadata/SEO
    page.jsx         Client entry — LanguageProvider + all sections
    globals.css      Design tokens + all styling
  components/        Hero, About, StatsBand, Timeline, Skills, Education, Contact, …
  context/           LanguageContext.jsx (EN/HE + RTL)
  hooks/             useCountUp, useMediaQuery
  i18n/translations.js   All text (EN + HE) + profile/contact info
public/              linor.jpg, cv-en.pdf, cv-he.pdf, favicon.svg, portrait.svg
```

## Customize

| What | Where |
|------|-------|
| **Profile photo** | `public/linor.jpg`. Path/fallback set in `src/i18n/translations.js` → `profile`. |
| **All text (EN + HE)** | `src/i18n/translations.js` — every string, per language. |
| **Contact / links** | `profile` object at the bottom of `src/i18n/translations.js`. |
| **Colors / fonts / spacing** | CSS variables at the top of `src/app/globals.css` (`:root`). |
| **Résumé PDFs** | `public/cv-en.pdf` and `public/cv-he.pdf`. |

## Regenerating the Hebrew résumé PDF

The Hebrew PDF is generated by `scripts/make_he_pdf.py` (uses `fpdf2` with RTL
text shaping via `uharfbuzz`):

```bash
pip install fpdf2 uharfbuzz
python scripts/make_he_pdf.py   # writes public/cv-he.pdf
```

The English PDF (`public/cv-en.pdf`) is the LinkedIn export.

## Tech notes

- **Framework:** Next.js 14 App Router. The interactive tree is a Client
  Component (`'use client'`); the page is statically prerendered.
- **Fonts:** `next/font/google` — Playfair Display (display), Inter (body),
  Heebo (Hebrew), exposed as CSS variables.
- **Smooth scroll:** Lenis (disabled under `prefers-reduced-motion`).
- **Animations:** Framer Motion (`whileInView`, `useScroll`/`useTransform`);
  all motion respects reduced-motion.
- **i18n:** lightweight context in `src/context/LanguageContext.jsx`; switching
  language updates `<html dir/lang>` and swaps to the Heebo font for Hebrew.
  SSR-safe (no hydration mismatch — preference applied after mount).
- **No backend** — fully static.
